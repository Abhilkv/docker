create node and index.html file 

1)  docker pull mongo
2)  docker pull mongo-express
3) create a seperate network for our applications 
    docker network create <networkName>
4) docker run -p 27017:27017 -d -e MONGO_INITDB_ROOT_USERNAME=admin -e MONGO_INITDB_ROOT_PASSWORD=secret --name <mongoDBName> --net <networkName> mongo    // -e stands for env variable 
5) docker run -d -p 8081:8081 -e ME_CONFIG_MONGODB_ADMINUSERNAME=admin -e ME_CONFIG_MONGODB_ADMINPASSWORD=secret --net networkName --name mongo-express -e ME_CONFIG_MONGODB_SERVER=mongoDBName mongo-express 
    mongo-express will be available on localhost:8081
